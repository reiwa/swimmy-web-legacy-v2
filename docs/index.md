### As Easy as 1, 2, 3

```bash
# install
$ yarn

# start writing
$ yarn vuepress dev

# build to static files
$ yarn vuepress build
```

::: warning COMPATIBILITY NOTE
VuePress requires Node.js >= 8.
:::
